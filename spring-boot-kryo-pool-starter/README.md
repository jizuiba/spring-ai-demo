自动管理实例的借用和归还
支持并发访问，无线程安全问题

对象池复用 Kryo 实例，避免频繁创建
支持预热（min-idle 配置）
LIFO 策略提高缓存效果
惰性初始化和按需扩容

3. 配置灵活

丰富的配置选项
支持 Spring Boot 配置属性
可自定义 Kryo 实例创建逻辑

4. 易于使用

自动配置，开箱即用
提供高级 API 和底层 API
工具类简化常用操作
完善的错误处理

5. 监控支持

池状态信息监控
详细的日志记录
性能指标收集
